TypeLess
by andriy-denysenko

An autoreplace utility allowing you to produce more text with less typing.

TypeLess is a utility that can help you save your time and effort when typing. It allows producing more text while typing less. TypeLess features are available from virtually any Windows application, so you can use the same text database in any window. An autoreplace feature helps you to accelerate your typing speed by expanding abbreviations. For example, you could assign a short abbreviation “imo” for a phrase “in my opinion”, and type 3 characters instead of 13 each time you need to type “in my opinion”.

Install

No installation required. As long as TypeLess requires no installation, just unpack the zipped file into a directory of your choice. Under that directory, a TypeLess directory is created where all files required for TypeLess to work will be located. For this release to work under Vista and later if unpacked under Program Files directory, you should grant write permissions for AppData directory located under TypeLess directory. To do this, right-click AppData directory, click Properties, open the Security tab in a dialog window that opens, and click Edit button. In a dialog Permissions for AppData that opens, select Users group in the Group or user names list, scroll down the Permissions for users list to find Write permission, and check a checkbox in Apply column. Then click OK to close Permissions for AppData dialog and OK to close AppData Properties dialog. Now you can launch TypeLess from the installation directory, and it will create shortcuts on your desktop in the Startup folder.
